# FIFA World Cup 2026 Result Simulator
 
## The Project
  
A web application designed for football fans to simulate the groups, match outcomes, and knockout stages of the historic 2026 FIFA World Cup. The project features an interactive and dynamic interface where users can predict scores, calculate group standings automatically, and trace the path of 48 national teams to the world title.

## Demo
Access the application: https://worldcupsimulator26.netlify.app/

## About the Project
The goal of this project is to provide an intuitive and complete simulation experience for the upcoming 2026 World Cup, reflecting its new official format. The application allows users to manipulate match results in real-time, automatically handling complex tournament rules such as group stage tie-breakers and the qualification of the best third-placed teams.

## Features

* Full simulation of the 48-team group stage (Groups A to L)
* Real-time standings updates (Points, Goal Difference, Goals Scored)
* Automatic calculation and ranking of the best third-placed teams
* Dynamic knockout bracket generation starting from the Round of 32
* Interactive penalty shootout simulator for knockout draws
* Save and load simulation progress via LocalStorage
* Fully responsive layout optimized for mobile and desktop screens


## Project Objectives

* Recreate the official FIFA World Cup 2026 tournament structure
* Demonstrate frontend logic for complex sports tournament rules
* Implement dynamic state management with local persistence
* Develop a fluid application using modern JavaScript/TypeScript
* Create a clean, engaging, and responsive sports user interface

## Technologies Used

* HTML5

## Application Structure
The application was organized into multiple views and independent modules, allowing for smooth navigation between the group stages and the final bracket. State management is handled entirely on the frontend, including:

* Real-time score tracking
* Group standings calculation
* Knockout bracket progression
* Local data persistence
* Dynamic UI updates
